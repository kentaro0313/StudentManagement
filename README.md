## プロジェクトの概要
### プロジェクト名:受講生管理システム



### プロジェクトの概要
様々な言語コースを受講する受講生を管理するシステムを作成しました。

受講生情報の検索、登録、更新と削除ができます。



## 画面動作



## 使用している主な技術
java/SpringBoot

MySQL

GitHub Actions(CI/CD)

AWS EC2

ALB



## 主要対応一覧
### 機能
・受講生情報の登録（全件検索、条件検索）

・受講生情報の検索

・受講生情報の更新

・受講生情報の削除（論理削除）
